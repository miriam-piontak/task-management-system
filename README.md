# Task Management System

A robust task management platform that enables users to organize their daily schedules efficiently. The system features user authentication, role-based task categorization, and flexible view options (list and calendar).

## 🚀 Key Features
* **User Authentication:** Secure login and registration for personalized access.
* **Task Management:** Create, view, and organize tasks tailored to your specific needs.
* **Dual View:** Toggle between a standard list view and a calendar-based interface for date-specific planning.
* **Admin Dashboard:** Administrators can define and manage master task categories, ensuring standardized task entry for all users.
